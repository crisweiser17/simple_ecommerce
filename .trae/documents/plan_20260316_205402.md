# Plano de Ação para Tradução Completa do Sistema

Após revisar o código do projeto linha por linha, identifiquei vários pontos onde textos fixos (hardcoded) ou variáveis dinâmicas do banco de dados (como nomes de categorias) ainda não estavam passando pelo sistema de tradução (`__()`), além de palavras que faltavam no nosso dicionário de traduções.

Abaixo está o plano detalhado para corrigir todos esses pontos:

## 1. Atualizar Dicionários de Idiomas (`lang/en.php` e `lang/pt.php`)
Vamos adicionar as chaves e traduções faltantes para cobrir todos os novos termos identificados, incluindo variações de capitalização (ex: "Add to cart" vs "Add to Cart"), status de pedidos e categorias:
- `'Add to Cart'` => `'Adicionar ao Carrinho'`
- `'Enter 6-digit code'` => `'Digite o código de 6 dígitos'`
- `'Email'` => `'E-mail'`
- `'No orders found.'` => `'Nenhum pedido encontrado.'`
- `'Order ID'` => `'ID do Pedido'`
- `'Status'` => `'Status'`
- `'Tracking'` => `'Rastreamento'`
- Status de pedidos: `'pending'` (pendente), `'ordered'` (realizado), `'paid'` (pago), `'shipped'` (enviado), `'completed'` (concluído)
- `'Category'` => `'Categoria'`
- `'Invalid token'` => `'Token inválido'`

## 2. Ajustar Templates do Usuário Final
Revisaremos os arquivos da pasta `templates/` para envolver todas as strings estáticas na função de tradução:

### `templates/layout.php` (Menu e Barra Superior)
1. Traduzir o placeholder da barra de pesquisa: de `placeholder="Search item..."` para `placeholder="<?php echo __('Search item...'); ?>"`.
2. Traduzir os links dinâmicos do menu (páginas do banco de dados) para que títulos em inglês possam ser traduzidos automaticamente se estiverem no dicionário.

### `templates/archive.php` (Página Inicial / Lista de Produtos)
1. Envolver os nomes dinâmicos das categorias exibidos na barra lateral e no título da seção na função `__()`, para que categorias como "Peptide Blends" sejam traduzidas corretamente para "Misturas de Peptídeos".

### `templates/product.php` (Detalhes do Produto)
1. Traduzir o texto "Category:" para usar o dicionário.
2. Envolver a categoria exibida (tanto no breadcrumb/trilha de navegação superior quanto nos detalhes da página) na função de tradução.

### `templates/login.php` e `templates/cart.php` (Login e Carrinho)
1. Traduzir o texto de preenchimento do input do token de login (de `placeholder="Enter 6-digit code"` para usar a tradução).

Dessa forma, absolutamente todo o conteúdo visível na navegação, desde os botões até os status do pedido e nomes de categoria, respeitará a escolha de idioma do usuário (Inglês ou Português). Posso iniciar as modificações no código?
