Vou implementar a persistência das abas utilizando o `localStorage` do navegador com o framework Alpine.js.

### Alterações em `templates/admin/dashboard.php`:

1.  **Inicialização do Estado (`x-data`)**:
    *   Vou alterar a definição inicial da variável `tab`. Em vez de fixar `'products'`, vou configurá-la para ler do `localStorage.getItem('admin_tab')` ou usar `'products'` como padrão.
    *   Vou adicionar um "watcher" (`$watch`) no Alpine.js para que, sempre que o valor de `tab` mudar, ele atualize automaticamente o valor no `localStorage`.

2.  **Opcional (Backup em PHP)**:
    *   Para garantir compatibilidade caso o JavaScript falhe ou para controle via URL, posso adicionar uma verificação de parâmetro GET (ex: `?tab=settings`), mas a solução via `localStorage` é a mais fluida para a experiência do usuário (SPA-like) sem recarregar a página para troca de abas.

Dessa forma, ao recarregar a página ou navegar de volta para o painel, a última aba acessada será restaurada automaticamente.
