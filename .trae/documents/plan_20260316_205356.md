Para atender ao seu pedido, farei as seguintes modificações para tornar o Banner totalmente customizável pelo painel Admin:

### 1. Adicionar Novos Campos no Painel Admin (Settings)
Vou editar o arquivo `templates/admin/dashboard.php` na seção "Banner Settings" para incluir:
- **Imagem à Direita (Opcional)**: Um campo para URL da imagem.
- **Opacidade do Fundo**: Um seletor deslizante (range) ou campo numérico de 0 a 100 para definir a opacidade da camada escura sobre a imagem de fundo.

### 2. Atualizar a Rota de Salvamento
Vou atualizar a rota `/admin/save-banner-settings` no arquivo `index.php` para capturar e salvar essas novas configurações no banco de dados:
- `banner_right_image_url`
- `banner_overlay_opacity`

### 3. Modificar o Frontend do Banner
Vou editar o arquivo `templates/archive.php` onde o banner é renderizado:
- **Imagem à Direita**: Adicionar uma lógica `if` do PHP. Se a configuração `banner_right_image_url` existir e não estiver vazia, exibiremos a imagem. Caso contrário, a `div` da imagem não será renderizada, e o texto poderá ocupar mais espaço ou ficar centralizado na imagem de fundo.
- **Opacidade**: Vou ajustar a classe CSS (Tailwind) responsável pela opacidade do overlay (`opacity-30` e `bg-gradient-to-r`). Usarei a configuração `banner_overlay_opacity` (com valor padrão se não existir) para injetar estilos inline (ex: `style="opacity: 0.X;"`) ou classes dinâmicas, permitindo que a camada escureça ou clareie a imagem de fundo conforme desejado.

Com isso, você terá controle total sobre o texto, imagens e aparência do banner diretamente pelo Admin. Posso prosseguir com a implementação?
